## Clone project
```
git Clone https://github.com/abugi/shuttlers-test.git
```

## Install project dependencies
```
npm install
```

### Run project on local development server
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
